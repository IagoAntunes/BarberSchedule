Pasta: Services
Descrição:
Esta pasta contém os projetos que implementam os endpoints da API e a lógica de negócios da aplicação. Cada serviço representa um domínio específico da aplicação e é responsável por expor os endpoints necessários e executar a lógica associada.

Componentes Principais:

Barbearia Service: Este projeto implementa os endpoints relacionados à gestão das barbearias, como cadastro, atualização de informações, e consulta de detalhes. Ele gerencia os dados específicos das barbearias e fornece a lógica de negócios necessária.
Orders Service: Este projeto implementa os endpoints relacionados ao agendamento de horários. Ele gerencia os dados de agendamentos e a lógica de negócios associada ao processo de agendamento e confirmação.
Shared Components: Componentes e utilitários compartilhados entre diferentes serviços, como modelos de dados, helpers, e extensões.
API Controllers: Implementação dos controladores de API que expõem os endpoints públicos para os aplicativos cliente e barbearia.
